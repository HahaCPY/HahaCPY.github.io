向量的基本運算規則

定義： 具方向性和大小且滿足平行四邊形法則的幾何對象 純兩： 單純數字，不具有方向性 規則：

  1. 加法運算： 唯有相同維度的向量可，不同維度或向量與純量不可
  2. 交換律與結合律： 相同維度向量成立
  3. 零向量： 一個向量的每個元素都為零
  4. 反向量： 與某個向量相加後會變成零向量，減法為反向量加法
  5. 向量與純量相乘： 向量的每個元素都須乘以純量
  6. 分配律： 向量相加再乘以純量和乘以多個純量時成立
  7. 向量1 = 原向量、向量-1 = 反向量、向量*0 = 零向量
